Qual das opções abaixo representa uma abordagem mobile-first na responsividade?

  1. Usar media queries apenas para resoluções maiores que 1024px.
  2. Desenvolver o layout primeiramente para desktops e depois ajustá-lo para dispositivos móveis.
  3. Utilizar frameworks como Bootstrap sem fazer modificações de responsividade.
  4. Iniciar o design pensando primeiramente em telas menores e usar media queries para ajustá-lo em telas maiores.
  5. Aplicar flexbox ou grid diretamente para telas grandes sem media queries.

Resposta Correta: Opção 4

A abordagem mobile-first consiste em iniciar o desenvolvimento do layout pensando em dispositivos móveis, geralmente com telas menores, e utilizando media queries para adaptar o design para telas maiores, como desktops e tablets.
